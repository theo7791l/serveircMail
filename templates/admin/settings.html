Comment configurer la réception dans Resend ?
  1. Va sur resend.com/domains et clique sur ton domaine
  2. Descends jusqu'à la section "Inbound" (ou "Receiving")
  3. Clique sur "Add Inbound Address"
  4. Dans "Endpoint URL", colle l'URL ci-dessus
  5. Ajoute les enregistrements MX indiqués par Resend dans ton DNS
  6. Copie le Signing Secret affiché et colle-le dans le champ ci-dessous
  7. Clique sur "Verify" dans Resend
Les DNS MX peuvent prendre jusqu'à 24–48h à se propager selon ton hébergeur DNS.
